PROJECT TITLE: Flappy Bird
PURPOSE OF PROJECT: A clone to the popular mobile game flappy bird.
VERSION or DATE: 2017/10/10
HOW TO START THIS PROJECT: Hits [Run] to start it.
AUTHORS: ianlizzo <ianliz22@icloud.com>
USER INSTRUCTIONS:

Click the mouse to flap the bird upward.
When the bird hits the pipes or the ground, the game is over.
